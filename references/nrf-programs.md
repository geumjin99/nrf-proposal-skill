# NRF Programs — Types, Eligibility, Budget, and the 2026 IRIS Naming

> Source basis: 2026 과기부 개인기초연구사업 1차 신청요강, 2025 인문사회분야 학술연구지원사업 종합계획,
> NRF 웹진, university 산학협력단 공지. **Always re-download the target 사업's 신청요강 before final
> submission** — figures, eligibility windows, and 2차 공모 details change yearly. URLs are in
> `references/academic-sources.md` and the README.

---

## 0. Two structural changes you MUST account for (2026+)

### Change A — submission system: e-R&D → **IRIS**
- 2025 and earlier: 이공 personal basic research applied through NRF's **e-R&D (ernd.nrf.re.kr)**;
  researcher records lived in **KRI (한국연구자정보)**.
- **2026 onward:** everything moves to **IRIS (범부처통합연구지원시스템, https://www.iris.go.kr)**,
  run by KISTEP. Researcher info moves to **NRI (국가연구자정보시스템)**, issuing a **국가연구자번호**.
- Prerequisites for an applicant: ① IRIS 회원가입 → ② NRI 연구자 전환 동의 (국가연구자번호 발급) →
  ③ register 학력/경력 + last-5-year 실적 in NRI (responsible PI must complete ③). IRIS 콜센터 1877-2041.

### Change B — 이공 program names were renamed
Maintain a **old↔new alias map** because users, universities, and most web pages still use the old
names.

| Old (≤2025, common usage) | New (2026+, IRIS) |
|---|---|
| 리더연구 | **리더연구** (유형B Top-tier, 유형A) |
| 중견연구 (창의·도약형) | **핵심연구** (유형C · 유형B · 유형A · 도약형) |
| 우수신진연구 | **신진연구** (유형B · 유형A) |
| 한우물파기 기초연구 | **핵심연구(한우물파기 기초연구)** |
| 세종과학펠로우십 | **세종과학펠로우십** (국내 / 국외연수 / 복귀·유치) |
| 생애첫연구 · 기본연구 | **기본연구 B · A** (안전망 성격, 2차 공모 비중) |

> When a user says "중견연구" treat it as 핵심연구; "우수신진" as 신진연구. Confirm the mapping out loud
> so they trust the routing.

## 1. 이공분야 (Science & Engineering) — 과기정통부 개인기초연구, NRF-run, IRIS

Budget figures below are **직접비 (direct cost)** per year; 간접비 is computed and paid separately by
NRF after selection (do not include it in the application budget).

| Program (2026 name) | Who it's for (eligibility highlights) | 기간 | 연 직접비 (상한) |
|---|---|---|---|
| **리더연구 유형B (Top-tier)** | top-tier PI, solo | 9y (5+4) | ≤ 12.8억 (1차년 9개월 ≤ 9.6억) |
| **리더연구 유형A** | top-tier PI | 9y (5+4) | ≤ 6.4억 (1차년 ≤ 4.8억) |
| **핵심연구 유형C** | mid-career (old 중견 창의형 high) | 1–5y | ≤ 2.4억 |
| **핵심연구 유형B** | mid-career | 1–5y | ≤ 1.6억 |
| **핵심연구 유형A** | mid-career (small) | 1–5y | ≤ 0.8억 |
| **핵심연구 도약형 유형2 / 유형1** | bridge for ending 중견/우수신진/후속 PIs | 3y | 유형2 ≤ 2.8억 / 유형1 ≤ 1.6억 |
| **핵심연구(한우물파기 기초연구)** | 박사 후 ≤15년 (학위 ≥2011.1.1), 국내대학 이공 | 10y (5+5) | ≤ 1.6억 |
| **기본연구 B / A** (2차 공모 중심) | research safety net (revived 2025) | ≤5y | B ≤ 0.7억 / A ≤ 0.5억 |
| **신진연구 유형B** (old 우수신진) | 박사 후 ≤7년 (학위 ≥2019.1.1) **또는** 만 39세 이하 (≥1986.1.1 출생), 국내대학 이공 전임교원/연구원 | 1–5y | ≤ 1.2억 |
| **신진연구 유형A** | same as 유형B | 1–3y | ≤ 0.8억 |
| **세종과학펠로우십 (국내트랙)** | 국내 박사, 박사 후 ≤7년 또는 만 39세 이하, **비전임** 연구자 | 5y | ≤ 1억 |
| **세종과학펠로우십 (국외연수)** | 해외 박사, ≤7년/39세 이하 | 1y | 인건비 0.7억 정액 + 간접비 350만 |
| **세종과학펠로우십 (복귀·유치)** | 해외 박사 귀국/유치 | 5y | ≤ 1억 |

Notes: 2025 expanded 중견 창의연구형 to ~885 grants and added 씨앗연구 for early-career; 2차 공모 is
mostly ~1억 small grants; 기본연구 was revived in 2025. The most common individual targets are
**신진연구 (early career)** and **핵심연구/중견 (mid-career)** — optimize the skill's defaults for these.

## 2. 인문사회분야 (Humanities & Social Sciences) — 교육부 학술연구지원, NRF-run

A **separate system** with different ministry (교육부), names, amounts, eligibility — and, crucially,
**blind review (무기명평가)**. 개인연구군:

| Program | Eligibility | 기간 | 경비 (per grant/yr) |
|---|---|---|---|
| **신진연구 (신진연구자지원)** | 조교수 임용 후 ≤5년, 또는 박사 후 ≤10년 | 1–3y | 2,000만 (간접비 별도) |
| **중견연구 (중견연구자지원)** | 조교수 임용 후 >5년, 또는 박사 후 >10년 | 1–10y 등 | 1,000만 또는 2,000만 |
| **우수학자 (우수학자지원)** | 박사 후 ≥15년 또는 조교수 경력 ≥10년, 대표 우수학자 | 5y (3+2) | 5,000만 |
| **학술연구교수 A (장기)** | 국내외 박사, 만 60세 이하, 비전임 | 5y (2+3) | 4,000만 |
| **학술연구교수 B (단기)** | 석사 이상, 만 64세 이하 | 1y | 2,000만 |
| **박사과정생 연구장려금** | 인문사회 박사과정생 | 2y (1+1) | 2,000만 |
| **석사과정생 연구장려금** (2025 신설) | 인문사회 석사과정생 | 1y | 1,200만 |

(공동연구, 번역·저술출판, 인문사회연구소, 글로벌인문사회융합연구, HK3.0 등은 집단/기관형 — 개인연구 아님.)

**Implication for the skill:** if the user is 인문사회, switch to the 교육부 system, enforce
**blind-review de-identification**, and pull the exact 평가표 from that program's 신청요강 (point
weights are listed per-program and are not the same as 이공).

## 3. Routing logic (how the skill should decide)

```
ask/infer 분야 ──┬── 이공 → IRIS + NRI; 신청요강(개인기초); NOT blind;
                 │ 실적 ≤5건/최근5년/질적우수성;
                 │ skeleton = 이공 6-section (proposal-structure.md §1)
                 └── 인문사회 → 교육부 system; BLIND review (de-identify);
                            program-specific 평가표; amounts per §2;
                            skeleton = 인문사회 (proposal-structure.md §2)

then map program name old→new (§0 Change B), confirm eligibility window
(연령 / 박사후 경과연수 / 직위), 기간, 연 직접비 상한, then set page limit
(proposal-structure.md) and score weights (evaluation-criteria.md).
```

Always confirm three things back to the user in plain language: **(1) which program (new + old name),
(2) page limit, (3) score weights** — then proceed.
